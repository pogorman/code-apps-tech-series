# Account Management Code App

A Power Platform Code App built with React + TypeScript for the Code Apps tech series. Full CRUD on Dataverse `account` table, deployed via `pac code push`.

## Prerequisites

- Node.js 20+
- [Power Platform CLI](https://learn.microsoft.com/en-us/power-platform/developer/cli/introduction) (`pac`)
- Active `pac auth` profile pointing at a Dataverse environment

## Quick Start

```bash
npm install
npm run build
pac code push
```

## Local Development

Two terminals required:

```bash
# Terminal 1 — Vite dev server (port 3000)
npm run dev

# Terminal 2 — Power Platform connection proxy
pac code run
```

Open the URL from `pac code run` (not Vite directly) — it wraps the app in the Power Platform host iframe so Dataverse calls work.

## Stack

| Layer | Tech |
|-------|------|
| Framework | React 19 + TypeScript 5.9 |
| Build | Vite 7 |
| Styling | Tailwind CSS v4 + shadcn/ui |
| Data | TanStack Query + generated Dataverse services |
| State | Zustand |
| Icons | Lucide React |
| Toasts | Sonner |

## Project Structure

```
src/
  generated/          # Auto-generated by pac code add-data-source (do not edit)
  components/
    ui/               # shadcn/ui primitives
    accounts/         # Account CRUD components
    layout/           # App shell
  hooks/              # TanStack Query hooks wrapping Dataverse services
  lib/                # Utilities (cn helper)
docs/                 # Tracked notes and research
.power/               # Power Platform schemas (auto-generated)
power.config.json     # Code App configuration
```

## Key Commands

| Command | Purpose |
|---------|---------|
| `npm run dev` | Start Vite dev server on port 3000 |
| `npm run build` | TypeScript check + production build to `dist/` |
| `pac code run` | Start Power Platform connection proxy |
| `pac code push` | Deploy to Power Platform |
| `pac code add-data-source -a dataverse -t <table>` | Register a Dataverse table |
