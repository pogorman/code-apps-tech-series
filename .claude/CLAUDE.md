# Project: Code Apps Tech Series

## What This Is

A Power Platform Code App demo for a tech series. React + TypeScript SPA deployed to Power Platform via `pac code push`. Full CRUD on Dataverse `account` table.

## Key Facts

- **Environment:** og-dv (`https://og-dv.crm.dynamics.com/`)
- **Auth profile:** `[2]` in `pac auth list` — `admin@M365x06150305.onmicrosoft.com`
- **CLI:** Use `pac code` for everything. `npx power-apps` does NOT exist.
- **No `$select`:** Omit `$select` from Dataverse queries — computed fields cause silent zero-row returns.
- **Generated code is read-only:** `src/generated/` and `.power/` are regenerated by `pac code add-data-source`.

## Stack

Vite 7 + React 19 + TypeScript 5.9 + Tailwind v4 + shadcn/ui + TanStack Query + Zustand + Lucide + Sonner

## Local Dev

Two terminals: `npm run dev` (port 3000) + `pac code run`

## Deploy

```bash
npm run build && pac code push
```

## Adding a Dataverse Table

```bash
pac code add-data-source -a dataverse -t <logicalname>
```

Then create hooks in `src/hooks/` and components in `src/components/<table>/`.
